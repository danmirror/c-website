## simple c using website

### prepare
- [x] sudo apt install make
- [x] sudo a2enmod cgi
- [x] sudo service apache2 restart

### run
- make clear
- make all
- cd ..
- sudo cp -R  c /usr/lib/cgi-bin/
- open your browser http://localhost/cgi-bin/c/index
